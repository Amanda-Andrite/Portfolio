# Portfolio Website

A personal portfolio website built with Svelte to show off my projects and skills.

## Features

* Responsive layout
* Interactive FAQ section
* Clean UI design

## Tech Stack

* Svelte / SvelteKit
* HTML, CSS, JavaScript

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Changes from High Fidelity Design
During development of the portfolio, several adjustments were made from the original High-Fidelity design due to time constraints and refinement of layout:

### Removed Features
- Removed the chibi character due to not enough time to continue 
- Did not include project images in my work for this current version  
- Back button navigation between pages was not implemented  

### Content Adjustments
- Added temporary filler text across sections.
- Updated project topics to specifcs(e.g. Wireframes & UX) to better reflect my personal work and interests  

### UI / Design Changes
- Simplified the contact card design to be more clean, minimal, and direct for the meantime 
- Adjusted the skills/qualifications section by centering it for better consistency with other pages  
- Improved the work/project cards layout by:  
  - Removing unnecessary text spacing mention in wireframe 
  - Adding hover effects for visual feedback
