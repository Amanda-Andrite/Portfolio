# Portfolio Website

A personal portfolio website built with Svelte to show off my projects and skills.

## Features

* Responsive layout
* Interactive FAQ section
* Clean UI design

## Tech Stack

* Svelte / SvelteKit
* HTML, CSS, JavaScript

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
